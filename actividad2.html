Estructura inicial


    El gatito # se utiliza como un indicador del preprocesador, la cual puede ser
    acompañado por el include (como en este caso), que se asegura que incluya una
    biblioteca.

    

    El stdio entre pico paréntesis (<stdio>), significa la biblioteca de donde 
    emergerán las funciones que se van a trabajar como printf, scanf, entre otros.

Firma de una función


    Se escribe la variable que va a devolver la función
    en este caso entero (int), ya que devuelve un 0.

    

    Se escribe main debido a que esta es la función principal.

    

    Se escribe el tipo de dato que va a recibir la función
    en este caso sería el vacío por lo que se puede o no escribir
    dentro del paréntesis.

Variables a trabajar


    Se utilizan las variables enteras o integer.

Funciones de entrada y salida


    Se utiliza la función de entrada scanf, la cual se utiliza
    para subir o escanear lo que se escribió por el usuario en la
    computadora, donde la f significa "format", lo que quedaría
    cómo escanear con formato.

    

    Se utiliza la función de salida printf, la cual se utiliza para
    imprimir o mandar a pantalla un mensaje del código, de igual manera
    la f significa "format", que en conjunto sería imprimir con formato.
Condicionales

    Se utiliza una condicional doble donde se pregunta si el usuario
    es mayor o menor de edad, según la edad que tenga el usuario.
    Esta es if y else.
    

    


#include <stdio>

int main (){
    int edad;
    printf("¿Cuántos años tienes? ");
    scanf("%d", &edad);

    if (edad >= 18) {
        printf("Eres mayor de edad.\n");
    } else {
        printf("Eres menor de edad.\n");
    }

    return 0;
}
